Tango on Baseball

§1. April 2, 2026

예측 타겟 방식
① Marcel 잔차 학습(Marcel + LSTM)
 Marcel이 이미 예측한 값을 베이스라인으로 두고, LSTM은 Marcel이 틀린 부분만 학습하는 구조. 


                                    실제 wOBA = 0.340
                                    Marcel 예측 = 0.320
                                    잔차 = +0.020  ← LSTM이 이걸 맞추려고 학습
 데이터 적어도 됨, 학습 안정적, Marcel의 통계적 안정성 그대로 활용. 단, Marcel 계산을 먼저 해야 함, Marcel의 구조적 한계를 못 벗어남.

② End-to-End LSTM

                                    [2019~2023 시즌 데이터] → LSTM → 2024 wOBA 직접 예측
 Marcel 의존 없음, 더 복잡한 패턴 자유롭게 학습 가능. 데이터 많이 필요, 과적합 위험, 학습 불안정할 수 있음.

③ Marcel 개선 → Marcel의 장점을 채용한 LSTM
 가중치, 나이 보정, 신뢰 가능한 예측 + 부상(갑작스러운 출전 경기 수의 변화를 기반으로 추정)
